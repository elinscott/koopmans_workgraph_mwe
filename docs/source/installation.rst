##############
 Installation
##############

The most recent release can be installed from `PyPI
<https://pypi.org/project/koopmans_workgraph_mwe>`_ with uv:

.. code-block:: console

    $ uv pip install koopmans_workgraph_mwe

or with pip:

.. code-block:: console

    $ python3 -m pip install koopmans_workgraph_mwe

*********************
 Installing from git
*********************

The most recent code and data can be installed directly from GitHub with uv:

.. code-block:: console

    $ uv pip install git+https://github.com/elinscott/koopmans-workgraph-mwe.git

or with pip:

.. code-block:: console

    $ python3 -m pip install git+https://github.com/elinscott/koopmans-workgraph-mwe.git

****************************
 Installing for development
****************************

To install in development mode with uv:

.. code-block:: console

    $ git clone git+https://github.com/elinscott/koopmans-workgraph-mwe.git
    $ cd koopmans-workgraph-mwe
    $ uv pip install -e .

or with pip:

.. code-block:: console

    $ python3 -m pip install -e .
